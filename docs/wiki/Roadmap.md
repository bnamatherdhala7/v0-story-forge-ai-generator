## v0.1 — Script generator (current)

**Status:** Working prototype. Async pipeline functional. Scripts generated. Video rendering is placeholder.

**What's delivered:**
- 3-screen UI (upload → processing → results)
- Async job pipeline with polling
- AI-generated hook titles + 15-second scripts
- n8n workflow for validation and logging
- 5 hook types: Pain Point, Transformation, Social Proof, Quick Win, Curiosity Gap

## v0.3 — Real video rendering

**Goal:** Creator gets downloadable `.mp4` files, not just scripts.

**Approach — Remotion (recommended):**
Remotion renders React components to video. The pipeline:
```
Script + Images + Brand Color
    ↓
Remotion React component:
  - Hook title card (0–3s)
  - Image montage with script overlay (3–12s)
  - CTA card with brand color (12–15s)
    ↓
Rendered .mp4 (9:16, 1080×1920)
```

**Alternative — Runway / Kling API:**
Feed images + script prompt → AI-generated video. Higher quality, higher cost (~$0.10–$0.50/video).

**Changes needed:**
- [ ] n8n "Generate Mock Videos" node → trigger Remotion render job
- [ ] Store rendered video files (Vercel Blob or S3)
- [ ] Real download URLs in results screen
- [ ] Progress bar reflects actual render time (not fake setTimeout)

## v0.5 — Multi-platform export

**Goal:** One generation run → downloads for every major platform.

| Format | Platform | Dimensions |
|---|---|---|
| 9:16 | TikTok, Instagram Reels, YouTube Shorts | 1080×1920 |
| 1:1 | Instagram feed, LinkedIn | 1080×1080 |
| 16:9 | YouTube intro, LinkedIn video | 1920×1080 |
| 4:5 | Instagram feed (recommended) | 1080×1350 |

Each format gets auto-resized layout: title card repositioned, image crops adjusted, captions reformatted.

## Pricing evolution

| Version | What's delivered | Suggested price |
|---|---|---|
| v0.1–v0.2 | Scripts only | Free / beta |
| v0.3 | Real video files | $29/month (20 generations) |
| v0.4 | Video + narration | $49/month (20 generations) |
| v0.5 | All formats + narration | $79/month (20 generations) |

At v0.3, AI costs per generation = ~$0.50–$1.00 (rendering). At $29/month for 20 runs, break-even is at ~$10–$20 in costs. Margin opens up significantly at higher plan tiers.
